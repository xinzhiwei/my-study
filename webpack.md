## 1.nrm ：通过了一些常用的npm包镜像地址
  + `安装nrm npm install nrm -g `
  + ` 查看nrm的镜像地址 nrm ls`
  + ` 带星号的代表当前使用的，切换 nrm use ++`
   ## 切换后 再使用npm install 下载的话，直接就会从你设置的地址下载
   ## nrm只是单纯的提供了几个常用的 下载包URL地址，并能够让在这几个地址之间进行切换，但是我们每次装包的时候，使用的装包感觉都是npm
